# Portfolio Architecture

A React + Vite portfolio structure with clear organization for reusable content and easy expansion.

## Project layout

- `src/main.tsx` — application entry point
- `src/App.tsx` — page structure and section arrangement
- `src/components/` — reusable UI sections and cards
- `src/data/` — project data and content sources
- `src/styles/` — global styling and design system foundation

## Getting started

1. Install dependencies
   ```bash
   npm install
   ```
2. Start the development server
   ```bash
   npm run dev
   ```
3. Build for production
   ```bash
   npm run build
   ```

## Recommended next steps

- Replace placeholder text and project items with your personal content.
- Add an `assets/` folder for images and logos.
- Create a `components/Testimonials.tsx` or `components/Resume.tsx` section if you want more pages.
- Use `src/data/projects.ts` as the single source of truth for portfolio entries.
